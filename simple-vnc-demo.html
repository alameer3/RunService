🚀 كيفية تشغيل المشروع كاملاً

للحصول على سطح مكتب Ubuntu كامل وفعّال، تحتاج لتشغيل المشروع باستخدام Docker:

# بناء حاوية Docker
docker build -t ubuntu-desktop .

# تشغيل الحاوية
docker run -p 5000:5000 -p 6080:6080 ubuntu-desktop

# الوصول عبر المتصفح
http://localhost:5000/vnc.html

كلمة المرور: 123456

البدائل المتاحة:

  • تشغيل الحاوية على خدمات السحابة
  • استخدام Docker Desktop
  • النشر على Replit مع دعم Docker
